  3. 03

    Governor — Budget-Aware Agent Loop for Claude Code

    2026

    A Claude Code plugin that closes an open loop: the model can see its context window but never the subscription quota clock. Governor puts the gauge in front of the model — a budget line injected every turn with a policy band attached — while a durability layer journals all work to disk and auto-checkpoints, so a hard mid-task cutoff costs one turn instead of the session.

    Limitations documented, not hidden: no hook point exists mid-generation, and quota data is Pro/Max-only — everything degrades gracefully to context-only mode.

    Node.js · Claude Code Plugin API · hooks · statusline · burn-rate regression

Under review — Quantum Machine Intelligence, Springer Nature · 2026

Bell-Decomposition Quantum Kernels: A Multi-Scale Benchmark on Earth Observation and Beyond

Kadam, P. B. (lead & primary corresponding author), Gaikwad, P. S., Raut, S. S., Shitole, T. U.

Introduces the Bell-decomposition family of two-qubit quantum-kernel feature maps (BSCM / SRQFM), unified under an adjustable prior, with two closed-form structural identities: a tight |α| ≤ ½ bound on every per-pair Pauli coefficient and exact ZZ-orthogonality under the uniform prior. Benchmarked across So2Sat LCZ42, EuroSAT, four UCI tabular sets, and a transverse-field Ising model.

macro-F1
+0.167 vs ZZ-PQK on So2Sat
significance
Holm-corrected Wilcoxon, p = 0.0078
scale
n = 16 qubits, N = 10,000
robustness
gap stable under depolarising noise

N.B. No quantum-advantage claim is made: tuned classical RBF-SVM ties or leads on every dataset tested. The within-quantum architectural gap is the contribution — reported honestly, consistent with current concentration theory.
